Project Overview

This project performs a complete survival analysis on data from a randomized controlled trial (RCT) comparing two concentrations of chlorhexidine mouth care (0.12% vs. 0.20%) for the prevention of Ventilator-Associated Pneumonia (VAP) among ICU patients.

The analysis focuses on evaluating:

Time-to-event outcomes

VAP-free survival probabilities

Differences between treatment groups

Clinical and statistical significance of predictors

The project uses Python (via lifelines, pandas, seaborn, matplotlib) to conduct rigorous survival modelling.
